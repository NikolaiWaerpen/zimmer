## VS Code

### Useful shortcuts:

<SyntaxHighlighter language="javascript">
  {`
    // Instant multi-line cursor on all instances of a highlight / current word
    cmd + shift + l
    // command palette
    cmd + shift + p
    // quick file palette
    cmd + p
    // multi-line cursor on word after word
    cmd + d
    // sort imports & removes unused imports
    option + shift + o
    // jump to specific section in current window
    cmd + shift + o
    // split current window
    cmd + 0
    // split current terminal
    cmd + shift + 0
  `}
</SyntaxHighlighter>
