## Vim

## Configuration:

- Replace "esc" to exit insert mode with j:

<SyntaxHighlighter language="typescript">
  {`
"vim.insertModeKeyBindings": [
    {
        "before": [
            "j",
            "k"
        ],
        "after": [
            "<esc>"
        ]
    }
] 
`}
</SyntaxHighlighter>

- Add vim-surround & vim-easymotion:

<SyntaxHighlighter language="typescript">
  {`
"vim.surround": true,
`}
</SyntaxHighlighter>

### Navigation:

<SyntaxHighlighter language="typescript">
  {`
// Move down
j 
// Move up
k
// Move left
h
// Move right
l
// Move x distance in any direction
[number][direction]
// First line of the file
gg
// Last line of the file
G
// Specific line of the file
:[number]
// Beginning of current line
0
// End of current line
$
// Move to matching parenthesis, bracket or space
%
// Top line on screen
H
// Middle line on screen
M
// Bottom line on screen
L
// The end of the current word
e
// Beginning of current word
b
// Beginning of next word
w
`}
</SyntaxHighlighter>

### Editing:

<SyntaxHighlighter language="typescript">
  {`
// Insert before current character
i
// Insert after current character
a
// Insert at the first non-whitespace character of the line
I
// Insert a line below the current line, then enter insert mode
o
// Insert a line above the current line, then enter insert mode
O
// Overwrite one character and return to command mode
r
// Undo
U
// Redo
ctrl + r
`}
</SyntaxHighlighter>

## Shortcuts:

<SyntaxHighlighter language="typescript">
  {`

3. enter
4. n for next instance, N for previous instance

// Highlight line from cursor (enter Visual mode)
v
`}
</SyntaxHighlighter>
